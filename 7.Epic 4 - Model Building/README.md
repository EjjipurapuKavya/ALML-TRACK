# Model Building

## Overview

The machine learning model is trained using the processed crop recommendation dataset.

## Algorithms Used

- Logistic Regression
- Decision Tree
- Random Forest

## Model Training

The dataset is divided into training and testing data. Different algorithms are trained and evaluated.

## Model Evaluation

Evaluation metrics include:

- Accuracy
- Precision
- Recall
- F1 Score

## Best Model

The model with the highest accuracy is selected for crop prediction.
