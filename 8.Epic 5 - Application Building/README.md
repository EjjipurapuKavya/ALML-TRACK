Th# Epic 5 - Application Building

## Overview

The OptiCrop application is developed using the Flask framework to provide crop recommendations based on soil and environmental parameters.

## Frontend

The frontend is developed using:
- HTML
- CSS

The user enters soil and climate information through a simple web interface.

## Backend

The backend is developed using Python and Flask.

It performs the following tasks:
- Accepts user input
- Loads the trained machine learning model
- Predicts the most suitable crop
- Displays the prediction result

## Features

- User-friendly interface
- Fast crop prediction
- Machine learning based recommendation
- Easy to use for farmers

## Technologies Used

- Python
- Flask
- HTML
- CSS
- Scikit-learn
- Pandas
- NumPy

## Output

The application recommends the best crop based on the given soil and environmental conditions.
